WDD 231: W05 Project Site Plan

By, Tanya Trampp

Site Name:

Kirtland Dojang

https://ttrampp.github.io/wdd231/dojang/index.html

This name was chosen because it is short and self explanatory.
it lets the user know that this site is in Kirtland, and about the Taekwondo dojang.
if someone in Kirtland is interested in Taekwondo, this would be an appeal to them.

Site Purpose:

This site provides information for someone who is looking to
learn Taekwondo and gain discipline, strength, and confidence.
The information given will provide a calendar of events, regular class information,
study information, and contact information.

Scenarios:

Where could I go to learn Taekwondo?
How much does it cost?
Who can I talk to to get more information?
Where can I get the name and meanings of the forms so I can study?

Color Schema:

The colors chosen are intended to match the Dojang's patch colors.
1-- Header #004494
2-- Background #C0C0C0
3-- Accent #f22b29

Typography

The font chose is Noto-Sans. It is a clean, clear read,
that shows professionalism with a dash of fun.
